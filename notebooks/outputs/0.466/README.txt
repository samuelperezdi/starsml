README
=======================

This folder contains the output CSVs generated by our cross-matching pipeline for two survey regions:
  • carina
  • coup

Each region has its own subfolder and exports one CSV per match‐case type.

Folder structure
----------------
data/
  carina/
    carina_N_yNWAY_trs{p_tr}.csv
    carina_N_OK_trs{p_tr}.csv
    carina_N_NoML_trs{p_tr}.csv
    carina_N_FLIP_trs{p_tr}.csv
    carina_N_NONE_trs{p_tr}.csv
  coup/
    coup_N_yNWAY_trs{p_tr}.csv
    coup_N_OK_trs{p_tr}.csv
    coup_N_NoML_trs{p_tr}.csv
    coup_N_FLIP_trs{p_tr}.csv
    coup_N_NONE_trs{p_tr}.csv

File naming
-----------
• `{region}` is either `carina` or `coup`  
• `{case}` is one of:
    – N_yNWAY   : all sources with NWAY match confidence (p_any > 0.5)  
    – N_OK      : NWAY‐confident sources whose ML label agrees (match_flag == 1)  
    – N_NoML    : NWAY‐confident sources with no ML matches (label == 0)  
    – N_FLIP    : NWAY‐confident sources whose ML match_flag disagrees (label == 1 but match_flag != 1)  
    – N_NONE    : sources with no NWAY confidence and no ML matches  
• `trs{p_tr}` tags the particular trained threshold value used  

Each CSV contains one row per Gaia candidate in that case.

Columns in every CSV
--------------------
  csc21_name           – unique Chandra source identifier  
  csc21_ra, csc21_dec  – Chandra coordinates (degrees)  
  min_theta_mean       – mean off‐axis angle (arcsec)  
  detect_stack_id      – stack identifier matching observation set  
  gaia3_source_id      – Gaia DR3 source identifier  
  p_i                  – individual match probability from NWAY  
  p_any                – summed match probability per Chandra source  
  p_match_ind          – ML “match‐indicator” score  
  separation           – angular separation (arcsec)  
  match_flag           – binary flag from NWAY best‐match selection  
  label                – ML ground‐truth label (0 = no match, 1 = match)  
  phot_g_mean_mag      – Gaia G‐band mean magnitude  
  phot_bp_mean_mag     – Gaia BP‐band mean magnitude  
  phot_rp_mean_mag     – Gaia RP‐band mean magnitude  
  bp_rp                – BP – RP colour index
